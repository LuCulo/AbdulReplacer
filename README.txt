How to use this tool, the super simple guide:

1) Download this repository as a .zip file 
2) Unpack the zip file to a folder
3) To manually change the input strings and output string, open getPagesSource.js in a text editor and modify the lines 12 and 13 to your liking
4) Go to chrome://extensions/ in your chrome browser and check the developer checkbox (top of the page)
5) Click "Load unpacked extension..." near the top of the page and find the folder that you created containing the repository files
6) Click OK

The extension will now be loaded on your chrome browser.  Delete the repository and re-use
steps 3-6 to change the input strings again.
